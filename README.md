Мета проєкту
Метою створення даного сайту є розробка простого та зручного веб-додатку для перегляду та вибору цифрової техніки. Сайт демонструє базові принципи побудови системи онлайн-торгівлі — від перегляду каталогу до імітації додавання товару в кошик.

Це тестовий навчальний проєкт, створений у межах курсової роботи. У поточній версії система є повністю клієнтською (HTML, CSS, JavaScript) і не підключена до онлайн-бази даних — усі дані статичні та завантажуються разом зі сторінкою.
